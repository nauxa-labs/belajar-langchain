## Pendekatan Pembelajaran

Untuk setiap proyek:

1. **Baca requirements** - pahami apa yang dibangun
2. **Review architecture** - lihat diagram & flow
3. **Implementasi step-by-step** - ikuti code examples
4. **Experiment** - modifikasi dan improve
5. **Deploy** (opsional) - gunakan LangServe
